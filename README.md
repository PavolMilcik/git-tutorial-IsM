## Code snippets - Some commands that I have learned and used on various examples:

### Initial settings:

```
$ git config
$ git init
```

### Working in a local repository:
```
$ git status
$ git add
$ git commit
$ git log
$ git branch
$ git merge
$ git rm
$ git mv
$ git switch
$ git checkout
$ git restore
$ git reset
$ git tag
```

### Working with remote repositor:
```
$ git remote
$ git push
$ git clone
$ git fetch + $ git merge
$ git pull
```
